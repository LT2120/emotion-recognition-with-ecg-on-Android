# emotion-recognition-with-ecg-on-Android

一个基于心电信号（ECG）完成情绪识别任务的系统。

该系统部署于安卓app中，使用深度学习的神经网络模型CNN，对从蓝牙设备收集到的心电信号进行情绪识别。

Android studio平台中完成app，pytorch完成神经网络模型训练。
